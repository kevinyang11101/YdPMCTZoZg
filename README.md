# 健康饮食管理系统 java203

## 项目概述

健康饮食管理系统是一个基于Java和Vue3的前后端分离的web平台，旨在提供健康饮食方案管理、数据统计可视化、在线交流等功能，适合作为Java项目实训及web网站平台设计与开发的参考。

## 技术栈

- 后端：Java，Spring Boot，MyBatis
- 前端：Vue3，ElementUI，HTML + JS + CSS
- 数据库：MySQL
- 项目管理工具：Maven
- 运行工具：JDK，IDEA，Navicat，Node.js

## 功能模块

### 用户角色

- 注册与登录
- 在线反馈咨询
- 饮食方案查看
- 新闻资讯浏览
- 论坛交流：发帖与评论
- 个人信息修改
- 饮食记录添加
- 健康报告查看
- 健康数据添加
- 帖子管理

### 营养师角色

- 注册与登录
- 数据统计可视化
- 饮食方案管理
- 健康数据管理
- 健康报告上传
- 饮食记录查看
- 在线反馈回复

### 管理员角色

- 数据统计可视化
- 饮食方案管理
- 健康数据管理
- 轮播图与新闻资讯管理
- 健康报告管理
- 在线反馈回复
- 论坛管理：增删改查，评论查看
- 用户管理
- 营养师管理

## 系统角色

1. 环境搭建：配置JDK，MySQL，Node.js等运行环境。
2. 数据库设置：创建数据库，导入SQL文件。

5. 运行项目：前后端分别启动。
